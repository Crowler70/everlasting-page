# Everlasting Page - Wedding Invitation Website

A theatrical-themed wedding invitation website built with React, TypeScript, and Tailwind CSS. The design features an elegant theater/cinema aesthetic with opening curtains animation, creating an immersive experience for guests.

## Features

- **Curtain Opening Animation** - Dramatic curtain reveal on page load
- **Interactive Guest Experience** - Smooth scroll reveal of wedding details
- **Countdown Timer** - Live countdown to the wedding date
- **Venue Information** - Location details with directions
- **Menu Card** - Dinner menu display
- **Dress Code Guidelines** - Attire recommendations
- **Gift Registry** - Gift information and wishlist
- **Transportation** - Transport details and parking info
- **RSVP Form** - Guest response collection with form validation
- **Responsive Design** - Works on mobile and desktop devices

## Tech Stack

- **Framework**: React 18 with TypeScript
- **Build Tool**: Vite
- **Styling**: Tailwind CSS with shadcn/ui components
- **Animations**: Framer Motion
- **Forms**: React Hook Form with Zod validation
- **Routing**: React Router DOM
- **Testing**: Vitest + Playwright

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Run tests
npm run test
```

## Project Structure

- `src/components/wedding/` - Wedding-specific components
- `src/components/ui/` - Reusable UI components
- `src/pages/` - Page components
- `src/hooks/` - Custom React hooks

## Customization

Edit the wedding details in the component files located in `src/components/wedding/` to customize names, dates, venue information, and other content.
